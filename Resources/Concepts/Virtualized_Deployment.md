# Virtualized Deployment

A method of deploying applications on Virtual Machines (VMs), which run on a single physical server, providing isolation and better resource utilization than traditional deployment.

*   Virtualized deployment era: As a solution, virtualization was introduced. It allows you to run multiple Virtual Machines (VMs) on a single physical server's CPU.

*   Virtualization allows applications to be isolated between VMs and provides a level of security as the information of one application cannot be freely accessed by another application.
