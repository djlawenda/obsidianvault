# Virtual Machine (VM)

A software-based emulation of a physical computer system, capable of running its own operating system and applications, isolated from other VMs on the same host.

*   A node may be a virtual or physical machine, depending on the cluster.

*   When running in a cloud environment and whenever a node is unhealthy, the node controller asks the cloud provider if the VM for that node is still available.



*   Virtualization allows you to run multiple Virtual Machines (VMs) on a single physical server's CPU.

*   Each VM is a full machine running all the components, including its own operating system, on top of the virtualized hardware.
