# kubectl

The command-line tool for Kubernetes, allowing you to run commands against Kubernetes clusters, deploy applications, and manage cluster resources.

## Excerpts & Key Information


> From: [[Processed_Transcripts/Kubernetes Nodes.md]] (Added on 2026-03-01 16:43)

*   You can create and modify Node objects using kubectl.

*   To mark a Node unschedulable, run: `kubectl cordon $NODENAME`

*   You can use `kubectl` to view a Node's status and other details: `kubectl describe node <insert-node-name-here>`
